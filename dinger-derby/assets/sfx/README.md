# Bat / park sound effects

## Included WAVs

Shipped modal **wooden bat** cracks (synthesized, not ripped from MLB):

| File | Use |
|------|-----|
| `bat_crack.wav` | Barrel / best contact |
| `bat_crack_solid.wav` | Solid square-up |
| `bat_crack_soft.wav` | Mishit / handle |
| `crowd_pop.wav` | Optional crowd surge (generated if missing) |

Replace any of these with your own **royalty-free** or self-recorded WAVs.

The game searches `assets/sfx/` relative to cwd (`build/../assets/sfx` works).

## Important: MLB audio

**Do not rip audio from MLB.TV, broadcasts, or official game footage.**

Use your own cage recordings or licensed SFX packs only.
